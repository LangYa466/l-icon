# L-Icon

基於各大品牌官方 Logo 修改的暗色主題 SVG 圖標集合，專為深色背景優化。部分圖標在原始基礎上進行了色彩調整，以確保在暗色主題下的可讀性。

## 如何使用

透過 GitHub Raw URL 直接引用：

```
https://icon.langya.io/resources/{filename}.svg
```

把 `{filename}` 替換成下方表格中的文件名就行，例如：

```html
<img src="https://icon.langya.io/resources/ovhcloud.svg" alt="OVHCloud" height="40">
```
